# Userscripts
Collection of Tampermonkey/Greasemonkey scripts for the browser.

### Template
[name].user.js
```js
// ==UserScript==
// @name           Script Name
// @namespace      https://mywebsite.com/
// @description    Script Description
// @include        *
// @author         Author Name
// @version        1.0
// @downloadURL    Raw JS Link
// @updateURL      Raw JS Link
// ==/UserScript==
```
